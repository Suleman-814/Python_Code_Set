
**Guess the Number Game (Python)**
A simple command-line game built using Python where the player tries to guess a randomly generated number within a given range.

**Description**
This is a beginner-friendly Python project that demonstrates:
* Random number generation
* User input handling
* Conditional statements
* Loops
The program selects a random number, and the player keeps guessing until they find the correct number. After each guess, the program provides hints.

**Features**
* Random number generation
* User-friendly prompts
* Feedback on guesses (Too high / Too low)
* Counts number of attempts
* Simple and easy to understand code

**Requirements**
Python 3.x
No external libraries are required (uses built-in modules only).


**How to Play**
1. The program generates a random number (e.g., between 1 and 100).
2. Enter your guess in the terminal.
3. The program will tell you:
   "Too low" if your guess is smaller
   "Too high" if your guess is larger
4. Keep guessing until you find the correct number.
5. The game will display the number of attempts taken.


**Example Output**
Guess a number between 1 and 100: 50
Too high!
Guess a number: 25
Too low!
Guess a number: 37
Congratulations! You guessed it in 3 attempts.
